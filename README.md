## 🚀 Major Update: The Gecko Era
JusBrowse has officially migrated from the native Android WebView to the **GeckoView** engine (the same engine powering Firefox). 
- **True Stealth**: GeckoView allows for deeper engine-level privacy controls that were impossible with standard WebView.
- **Enhanced Fingerprinting Defense**: Integrated support for `resistFingerprinting` and advanced WebExtension based blocking.
- **Improved Isolation**: Hardware-level and engine-level isolation for different personas.
